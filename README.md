# Hotel Barborka - Booking Management System

A Python-based reservation management prototype designed for **Hotel Barborka** in Přední Výtoň. This project demonstrates backend development fundamentals, object-oriented programming (OOP), and SQL database integration.

## Key Features
* **Object-Oriented Design:** Guest data is managed through structured Python classes.
* **Database Integration:** Implements SQLite for persistent data storage.
* **SQL Logic:** Includes advanced querying techniques such as `LIKE` operators and parameterized queries to prevent SQL injection.
* **Business Context:** Developed as a solution for a real-world hospitality business located near the Lipno Dam.

## Technical Skills Demonstrated
* **Language:** Python 3.x
* **Database:** SQL / SQLite
* **Tools:** Git / GitHub
* **Security:** Prevention of SQL injection via query parameterization.

## How to Run
1. Clone the repository.
2. Run `python main.py` in your terminal.
3. The system will automatically create `barborka.db` and populate it with sample guest data.
